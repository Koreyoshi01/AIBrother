# 二氧化碳吸收与解析实验操作手册

## 实验目的
模拟工业上化学吸收法捕集CO2，使用碱性溶液吸收混合气体中的CO2，并通过加热或减压实现解析再生。

## 实验原理
CO2与碱性吸收剂发生化学反应：
- MEA法：CO2 + 2HOCH2CH2NH2 → HOCH2CH2NHCOO- + HOCH2CH2NH3+
- NaOH法：CO2 + 2NaOH → Na2CO3 + H2O

## 所需试剂与材料

### 吸收剂
1. **单乙醇胺（MEA）** — 最常用的醇胺类吸收剂，反应速率快，吸收容量高
   - 推荐浓度：**30 wt%**（质量分数）
   - 配制：30g MEA + 70g 去离子水
2. **NaOH溶液** — 传统强碱吸收剂
   - 推荐浓度：**1-2 mol/L**
3. **MDEA（N-甲基二乙醇胺）** — 选择性好，再生能耗低
   - 推荐浓度：**40-50 wt%**

### 气体
- 模拟烟气：10-15% CO2 + 85-90% N2（或空气）
- 高纯N2：用于吹扫和稀释

### 辅助试剂
- 去离子水 — 配制溶液和洗涤
- pH指示剂（酚酞）— 检测吸收饱和程度
- 盐酸标准溶液（0.1 mol/L）— 用于滴定分析吸收液浓度

## 实验装置

1. **CO2钢瓶 + 质量流量计** — 控制气体流量
2. **鼓泡吸收反应器 / 填料塔** — 气液接触
3. **恒温水浴** — 控制吸收温度（推荐20-40°C）
4. **CO2分析仪 / 气相色谱** — 检测进出口气体组成
5. **蠕动泵** — 循环吸收液
6. **冷凝回流装置** — 解析时减少吸收剂损失

## 标准操作流程

### 吸收实验
1. 配制30 wt% MEA溶液，倒入吸收反应器
2. 设置水浴温度至 **30°C**，预热15min
3. 通入N2吹扫装置5min，排尽空气
4. 调节CO2和N2流量，使CO2体积分数为**12%**，总流量**1 L/min**
5. 气体从反应器底部通入，与吸收液逆流接触
6. 每隔5min记录进出口CO2浓度，计算吸收效率
7. 实验持续至出口CO2浓度稳定（吸收饱和）

### 解析实验
1. 将富液转移至解析反应器
2. 加热至 **100-120°C**，常压解析
3. 冷凝回流，收集解析出的CO2
4. 解析时间约30-60min
5. 测定贫液中CO2负载量，计算解析效率

## 注意事项
- MEA有腐蚀性，操作时戴手套和护目镜
- CO2浓度较高时注意通风
- 吸收液温度不宜超过50°C（高温会导致MEA降解）
- 实验后废液统一收集处理，不可直接排放
